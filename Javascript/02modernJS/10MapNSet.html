Map과 Set의 도입 시기 Map과 Set은 ES6 (ECMAScript 2015)에서 도입됨. 기존 Object와 Array의 한계를 보완하기 위해 추가된 새로운 데이터 구조임. 📌 Map vs Set 개요 구조 도입 버전 주요 특징 Map ES6 (2015) 키-값(key-value) 구조, 키에 객체 사용 가능 Set ES6 (2015) 중복 없는(unique) 값들의 집합 📌 Map (키-값 저장) 키로 객체도 사용할 수 있음 (기존 Object는 문자열 키만 가능) 순서를 유지함 js 복사 편집 const map = new Map(); map.set('name', 'Alice'); map.set(42, 'Age'); map.set({ key: 'value' }, 'Object Key'); console.log(map.get('name')); // "Alice" console.log(map.size); // 3 📌 Set (중복 없는 값 저장) 중복된 값이 자동으로 제거됨 순서를 유지함 js 복사 편집 const set = new Set([1, 2, 3, 3, 4, 4]); console.log(set); // Set { 1, 2, 3, 4 } set.add(5); set.delete(2); console.log(set.has(3)); // true console.log(set.size); // 4 ✅ 즉, Map과 Set은 ES6에서 도입된 새로운 데이터 구조로, 기존 Object와 Array의 단점을 보완하는 역할을 합니다! 🚀 📌 Map 예제 (키-값 쌍) js 복사 편집 // Map을 사용하여 학생 정보 저장하기 const studentMap = new Map(); // 학생 정보 추가 studentMap.set('Alice', { age: 20, grade: 'B' }); studentMap.set('Bob', { age: 22, grade: 'A' }); studentMap.set('Charlie', { age: 23, grade: 'C' }); // Map에서 값 가져오기 console.log(studentMap.get('Alice')); // { age: 20, grade: 'B' } console.log(studentMap.get('Bob')); // { age: 22, grade: 'A' } // Map의 크기 console.log(studentMap.size); // 3 // Map 순회하기 studentMap.forEach(function(value, key) { console.log(`${key} -> Age: ${value.age}, Grade: ${value.grade}`); }); // 키 존재 확인 console.log(studentMap.has('Charlie')); // true console.log(studentMap.has('David')); // false // Map에서 키-값 쌍 삭제 studentMap.delete('Alice'); console.log(studentMap.size); // 2 📌 Set 예제 (중복 없는 값) js 복사 편집 // Set을 사용하여 중복 없는 값 저장하기 const numberSet = new Set([1, 2, 3, 4, 4, 5, 5]); // Set의 크기 console.log(numberSet.size); // 5 (중복 값은 제외됨) // Set에 값 추가 numberSet.add(6); console.log(numberSet); // Set { 1, 2, 3, 4, 5, 6 } // Set에서 값 삭제 numberSet.delete(3); console.log(numberSet); // Set { 1, 2, 4, 5, 6 } // Set에서 값 존재 확인 console.log(numberSet.has(2)); // true console.log(numberSet.has(3)); // false // Set 순회하기 numberSet.forEach(function(value) { console.log(value); }); 📌 예제 설명 Map 예제: Map은 키-값 쌍으로 데이터를 저장합니다. set() 메서드를 사용하여 데이터를 추가하고, get() 메서드를 통해 값을 조회합니다. size 프로퍼티로 Map의 크기를 알 수 있고, forEach()를 이용하여 순회할 수 있습니다. has()와 delete() 메서드로 키 존재 여부 확인 및 삭제가 가능합니다. Set 예제: Set은 중복을 허용하지 않는 값을 저장합니다. add() 메서드로 값을 추가하고, 중복된 값은 자동으로 제거됩니다. has()를 통해 값의 존재 여부를 확인하고, delete()로 값을 삭제할 수 있습니다. forEach() 메서드를 사용하여 Set의 값을 순회할 수 있습니다.
